64 Pad Explorer

64パッド上のスケール・コード・ボイシングを探索するブラウザツール。
インストール不要 — ブラウザだけで動きます。

アプリを開く

特徴

  • インストール不要 — ブラウザだけで動く可視化ツール。MIDI入出力、コード判定、ボイシング自動保持に対応
  • パッドを弾けなくても使えます — コードを3ステップで入力。テンションまで表示できます。パッドをクリックでも入力可能
  • テキストコード入力 — 「/」キーでCm7, G7(b9,#11), Am7/G等をテキスト入力。オートコンプリート候補表示。dim, h(=m7♭5)等の略記法にも対応
  • コード判定 — パッド上でクリックした音からコード名をリアルタイム判定。omitやオンコード、複数候補表示に対応
  • ギター・ベース・鍵盤表示 — ギターフレットボード、ベース、ピアノ鍵盤にコード構成音を表示。ギターの手形入力でパッドにも自動反映されます。Vキーでギターライク（1行1音）/ コンパクト配置を切り替え可能
  • スケール表示 — 31スケール × 12キー。ダイアトニック、ハーモニックマイナー、メロディックマイナー、ペンタトニック、ビバップ等
  • ダイアトニックコード表示 — スケールに合わせたダイアトニックコードをバーに表示。クリックまたはショートカットキーで即座にコードを鳴らせます
  • ボイシング操作 — 転回形、Drop 2/3、Shell Voicing、Omit、Rootless。カーソル上下で転回形切替、ボタンで複数フィンガリング表示
  • コードメモリー機能 — 気に入ったコードを16スロットに保存。MIDIファイルとして書き出してDAWに送れます
  • Performモード — メモリーに記録されたコードを、16パッドまたはPCキーボードでリアルタイム演奏できます
  • Available Scale逆引き — 入力したコードが属しうるスケールを逆引き表示。テンション候補も一目でわかります。Practical/Diatonicソート切替対応
  • ピボットコード思考 — 手動でコードを入力すると、ダイアトニックバーが消え「このコードがどのキーに属しうるか」を一覧表示。転調の分析に最適
  • テンション実用性表示 — ドミナント7以外のコードでは、実践で使われにくいテンション（b9, #9, b13等）が薄く表示されます。選択は可能です
  • 五度圏表示 — Circle of Fifths を表示。キー選択と連動し、ダイアトニックコード・機能（T/SD/D）を表示。マイナースケールモード切替対応
  • メモリーバンク — 16スロット×最大16バンクでコード進行を管理。バンクの切替・複製・リネームに対応。[ ]キーで切替
  • オートフィルター — エンベロープフィルター（Auto-Wah）。Cry Baby / Mu-Tron / Moog の3タイプ、周波数・レゾナンス・感度・ディケイを調整可能
  • ドライブ — アナログサチュレーション。温かみのある歪みをサウンドに追加
  • クラヴィネット音源 — E.PIANOに2種追加（Clav Muted, Clav Bright）。ファンクに最適
  • MIDI感度調整 — Push 3風の4パラメータ（Threshold, Drive, Compand, Range）でベロシティカーブを調整。カーブをリアルタイムに可視化。全MIDIコントローラー対応
  • Push 3オクターブ連動 — Push 3のオクターブ▲▼ボタンでアプリのオクターブシフトが自動連動
  • 多言語対応 — 9言語（EN / JA / ZH / ES / FR / PT / DE / KO / IT）

画面紹介

Scale Scaleモード

キーとスケールを選択すると、64パッド上にスケール音が色分けで表示されます。ダイアトニックバーからコードを選択できます。

C Major (Ionian) — ルート音(オレンジ)、スケール音(ブルー)、特性音(黄)で色分け

Chord Chordモード

Root → Quality → Tension の3ステップでコードを構築。ボイシングの転回・Drop・Shell・Omitも操作できます。Parent Scaleパネルでそのコードが属しうるスケールを逆引きできます。

Dm7 — コードトーン、3rd(ピンク)、7th(グリーン)を表示。ボイシングボックスで配置候補を確認

Input Inputモード

パッドを自由にクリックしてコード名をリアルタイム判定。理論を知らなくてもコードがわかります。

C7 — パッドをクリックするだけでコード名を自動判定。複数候補も表示

Performモード

メモリーに保存したコードを16パッドに配置し、キーボードやMIDIパッドでリアルタイム演奏できます。

Performモード — C7, Fm7, G7, Am7をスロットに保存。キーボード4×4グリッドで演奏

MIDI感度調整（Velocity Sensitivity）

MIDIコントローラーのベロシティ感度をPush 3風の4パラメータで調整できます。サウンドパネル内のVELOCITY SENSITIVITYセクションで設定。カーブがリアルタイムに可視化されます。

  • THRESH (Threshold) — 最低入力ベロシティ。これ以下のタッチは無音になります（0〜64）
  • DRIVE — カーブの立ち上がり。+で弱いタッチでも大きな音に、-で強く叩かないと鳴らない（-64〜+64）
  • COMP (Compand) — ダイナミックレンジ圧縮(+)/拡張(-)。+で音量差が均一に、-で強弱の差が広がる（-64〜+64）
  • RANGE — 最大出力ベロシティ。127で最大、下げると全体的に弱くなる（1〜127）

Push 3だけでなく、LinnStrumentや他のMIDIコントローラーでも使えます。設定はlocalStorageに自動保存されます。

Push 3 オクターブボタン対応

Push 3のオクターブ▲▼ボタン（CC#55/CC#54）をアプリ内のオクターブシフトに連動させます。Push 3本体のオクターブボタンを押すと、アプリの表示オクターブが自動的に切り替わります。

Available Scale + ピボットコード

コードが属しうるスケールを逆引き表示。Practical（実用性順）/Diatonic（理論順）の切替が可能。手動コード入力時はダイアトニックバーが消え、ピボットコードとして分析できます。

Dm7 → C Major (Dorian), C Mel.Min (Dorian b2), F Major (Aeolian) 等。Available Tensionsも表示

※ ペンタトニック/ブルーススケールはモード（横のアプローチ）であり、コード・スケール理論（縦の対応）とは別体系のため、Available Scaleでは扱いません。実戦ではペンタトニックが最初の選択肢になることが多いです。

TASTY Voicing（HPS限定）

129種類のプロフェッショナル・ボイシングを度数配列で表示。コードを選んで T キーで起動し、連打でレシピを巡回。パッドに全音が収まる最適ポジションを自動選択します。

info欄に度数の積み順（例: m9 (b3-b7-9-5-1)）が表示されます。これがボイシングの構造——下からどの順番で音を積むかを示しています。

ラベルの意味

  • Rootless — ルート（1度）を省略したボイシング。バンドではベーシストがルートを弾くため、鍵盤やパッドではルートを省いて上部構造だけを弾くのが一般的です。ジャズ・R&B・ネオソウルで多用されます。
  • Omit5 — 5度を省略。5度は響きの個性が薄いため、代わりにテンションを入れてサウンドを豊かにする実践的なテクニックです。
  • Omit3 — 3度を省略。sus系やクォータルな響きに近づきます。

操作

  • T — TASTY起動 / 次のレシピへ
  • Escape — 元のコードに戻る
  • Space — 現在のボイシングを再生
  • Opt+キー — メモリースロットに保存（1234/QWER/ASDF/ZXCV）

こんな時に使えます

  • 「このコードの押さえ方を調べたい」 → Chord モードでRoot・Quality・Tensionを選択 → ボイシングボックスで配置を確認
  • 「レッスンで出てきたスケールをパッド上で確認したい」 → Scale モードでキーとスケールを選択。31スケール対応
  • 「コード進行を作ってDAWに送りたい」 → コードをメモリースロットに保存 → MIDI Export でDAWにインポート
  • 「このコード、どのキーのダイアトニック？」 → Available Scaleパネルで逆引き。Practical/Diatonicソートで実用性順にも並べ替え可能
  • 「転調先のキーを探したい（ピボットコード）」 → Chord モードで手動コード入力。ダイアトニックバーが消え、Available Scaleがそのコードの属しうる全キーを表示します
  • 「コードを叩いて進行を確認したい」 → メモリースロットにコードを保存して、MIDIコントローラーやPCキーボードで叩いてみてください
  • 「ギターで押さえたコードをパッドで確認したい」 → ギターフレットボードをクリックして入力 → Vで「パッドに反映」→ ギターライク / コンパクト配置を選択
  • 「プロが使うボイシングを知りたい」 → Chord モードでコードを作り T でTASTY起動。129種のボイシングを度数付きで確認、メモリーに保存してDAWへ

最初にやること

  1. キーを選ぶ（例: C）
  2. スケールを選ぶ（例: Major）→ パッド上にスケール音が表示される
  3. ダイアトニックバーでコードをクリック（例: Dm7）→ コードトーンが表示される
  4. パッドをクリックして音を鳴らす。ボイシングボックスで配置を変更できます
  5. 気に入ったコードは Option+1234/qwer/asdf/zxcv でメモリーに保存（16スロット対応）
