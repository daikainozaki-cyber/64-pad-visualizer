64 Pad Explorer

64パッド上のスケール・コード・ボイシングを探索するブラウザツール。
音源不要 — Chromeだけで音が出ます。

アプリを開く

特徴

  • 音源不要 — Chromeだけで音が出ます。オルガン4種 + エレピ7種の音源を内蔵。エフェクト（リバーブ、フェイザー、フランジャー、トレモロ、フィルター）も搭載
  • パッドを弾けなくても使えます — コードを3ステップで入力。テンションまで表示できます。パッドをクリックでも入力可能
  • コード判定 — パッド上でクリックした音からコード名をリアルタイム判定。omitやオンコード、複数候補表示に対応
  • ギター・ベース・鍵盤表示 — ギターフレットボード、ベース、ピアノ鍵盤にコード構成音を表示。ギターの手形入力でパッドにも自動反映されます
  • スケール表示 — 31スケール × 12キー。ダイアトニック、ハーモニックマイナー、メロディックマイナー、ペンタトニック、ビバップ等
  • ダイアトニックコード表示 — スケールに合わせたダイアトニックコードをバーに表示。クリックまたはショートカットキーで即座にコードを鳴らせます
  • ボイシング操作 — 転回形、Drop 2/3、Shell Voicing、Omit、Rootless。カーソル上下で転回形切替、ボタンで複数フィンガリング表示
  • コードメモリー機能 — 気に入ったコードを16スロットに保存。MIDIファイルとして書き出してDAWに送れます
  • Performモード — メモリーに記録されたコードを、16パッドまたはPCキーボードでリアルタイム演奏できます
  • Parent Scale逆引き — 入力したコードが属しうるスケールを逆引き表示。テンション候補も一目でわかります
  • 多言語対応 — 9言語（EN / JA / ZH / ES / FR / PT / DE / KO / IT）

画面紹介

Scale Scaleモード

キーとスケールを選択すると、64パッド上にスケール音が色分けで表示されます。ダイアトニックバーからコードを選択できます。

C Major (Ionian) — ルート音(オレンジ)、スケール音(ブルー)、特性音(黄)で色分け

Chord Chordモード

Root → Quality → Tension の3ステップでコードを構築。ボイシングの転回・Drop・Shell・Omitも操作できます。Parent Scaleパネルでそのコードが属しうるスケールを逆引きできます。

Dm7 — コードトーン、3rd(ピンク)、7th(グリーン)を表示。ボイシングボックスで配置候補を確認

Plain Plainモード

パッドを自由にクリックしてコード名をリアルタイム判定。理論フィルタなしで探索 → 発見 → 理論を後から知るモードです。

C7 — パッドをクリックするだけでコード名を自動判定。複数候補も表示

Performモード

メモリーに保存したコードを16パッドに配置し、キーボードやMIDIパッドでリアルタイム演奏できます。

Performモード — C7, Fm7, G7, Am7をスロットに保存。キーボード4×4グリッドで演奏

Parent Scale + ギター表示

コードが属しうるスケールを逆引き表示。ギターフレットボードにもコードトーンが表示されます。

Dm7 → C Major (Dorian), C Mel.Min (Dorian b2), F Major (Aeolian) 等。Available Tensionsも表示

こんな時に使えます

  • 「このコードの押さえ方を調べたい」 → Chord モードでRoot・Quality・Tensionを選択 → ボイシングボックスで配置を確認
  • 「レッスンで出てきたスケールをパッド上で確認したい」 → Scale モードでキーとスケールを選択。31スケール対応
  • 「コード進行を作ってDAWに送りたい」 → コードをメモリースロットに保存 → MIDI Export でDAWにインポート
  • 「このボイシング、どのキーのダイアトニック？」 → Parent Scaleパネルで逆引き。Available Tensionsも一目でわかります
  • 「パッドで即興演奏の練習がしたい」 → Performモードでメモリースロットのコードを16パッド配置。キーボードまたはMIDIパッドで演奏
  • 「ギターで押さえたコードをパッドに転写したい」 → ギターフレットボードをクリックして入力 → パッドに自動表示 → コード名も判定

3つのモード

Scale スケールとダイアトニックコードを表示

Plain パッドを自由にクリック → コード名を自動判定

最初にやること

  1. キーを選ぶ（例: C）
  2. スケールを選ぶ（例: Major）→ パッド上にスケール音が表示される
  3. ダイアトニックバーでコードをクリック（例: Dm7）→ コードトーンが表示される
  4. パッドをクリックして音を鳴らす。ボイシングボックスで配置を変更できます
  5. 気に入ったコードは Shift+1〜0 でメモリーに保存
