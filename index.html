64 Pad Explorer

64パッド上のスケール・コード・ボイシングを探索するツール。

Guide / ガイド →

Scale Mode

  • Key — 12キーから選択
  • Scale — ダイアトニック、ハーモニックマイナー、メロディックマイナー、ペンタトニック、ビバップ等 31種
  • Diatonic Bar — 選択したスケールのダイアトニックコードを表示。クリックでChordモードへ

Chord Mode（3ステップ）

  • Step 1: Root — ピアノ鍵盤でルートを選択
  • Step 2: Quality — コードタイプを選択（Maj, m, aug, dim, sus 等）
  • Step 3: Tension — テンションを追加（9, b9, #9, 11, #11, 13, b13 等）。複数選択可
  • On Chord — 「/」ボタンでスラッシュコード（分数コード）を指定

Voicing

  • Omit 5 / Rootless / Omit 3 — 音を省略
  • Shell 1-3-7 / 1-7-3 — シェルボイシング（+1, +2で音を追加）
  • Inversion — 転回（Root, 1st, 2nd, 3rd）
  • Drop 2 / Drop 3 — ドロップボイシング
  • ボイシングボックス — パッド上の配置候補をタップで切り替え（脈動バッジ = 代替配置あり）

表示

  • Guitar / Piano / Staff — ギターフレットボード、ピアノ鍵盤、五線譜を個別にON/OFF
  • 音名/度数 — ギター・パッド上の表示を音名(C, D, E)と度数(R, 2, 3)で切り替え
  • Octave ▲▼ — パッドの音域をオクターブシフト

MIDI

  • WebMIDI対応デバイスを自動検出
  • Ableton Push のシリアルMIDI出力に対応
  • 入力した音からコードを自動判定

色の意味

  • Root — ルート音
  • Scale / Chord — スケール音・コードトーン
  • 特性音 — そのスケール/コードを特徴づける音
  • 3rd — ガイドトーン（3度）
  • 7th — ガイドトーン（7度）
  • Tension — テンション音
  • Avoid — コードトーンの半音上のスケール音（Chordモード時）

配色: Okabe-Ito（色覚多様性対応）

ギター・ピアノ入力

  • ギターフレットボード / ピアノ鍵盤をクリックして音を入力
  • 入力した音からコードを自動判定してパッドに反映
  • Clear — 入力をクリア、Play ▶ — 入力した音を再生

Input Mode

  • パッドをクリックして自由に音を選択 → コード名をリアルタイム判定
  • 理論フィルタなし。探索 → 発見 → 理論を後から知る
  • コード検出時にパッド上に度数ラベル（R, 3, 5, b7等）を表示。ギター・ベース・ピアノ・五線譜にも連動
  • Memory — 16スロットにコードを保存・呼び出し
  • MIDI Export — メモリーのコードをMIDIファイルとして書き出し

Perform Mode

  • Memory / Perform ボタンまたは p キーで切替
  • メモリースロットのコードを16パッド配置でリアルタイム演奏
  • キーボード（4×4グリッド）またはMIDIパッドで発音
  • スロットのドラッグ&ドロップで並び替え可能

キーボードショートカット

Scale / Chord モード

  • 1〜7 — ダイアトニックコード選択（7音スケール時）
  • a〜i — ボイシングボックス選択（代替配置の循環も可）
  • ↑↓ — 転回形の切り替え（Chordモード時）
  • ←→ — ルートを半音移動（同型コードのトランスポーズ）
  • o — Omit 5 切り替え
  • r — Rootless 切り替え
  • s — Shell循環（off → 1-3-7 → 1-7-3 → off）
  • d — Drop循環（off → Drop 2 → Drop 3 → off）
  • x — コードクリア（Chordモード）

Input モード

  • c — Capture 開始/次のスロットへ保存
  • e — Capture 終了
  • 1〜9, 0 — メモリースロット呼び出し（Edit）
  • ←→ — 半音移動（全ノート±1）
  • ↑↓ — 転回形（最低音↑1oct / 最高音↓1oct）
  • x — 全クリア
  • Chord切替 — 検出コードをChordモードに引き継ぎ

Perform モード

  • 1234 / qwer / asdf / zxcv — 4×4グリッドでスロット1〜16を発音
  • MIDIパッド（ノート36〜54）でも発音可能

共通（全モード）

  • p — Perform表示の切り替え
  • Cmd/Ctrl+z — メモリースロットのUndo（最大30回）
  • Shift+1〜0 — Inputメモリースロットに保存（どのモードからでも）
  • Esc — ボイシングボックス選択解除 / ヘルプを閉じる
  • ? — ヘルプ表示切替
